Seemantham RSVP Website

How to edit later:
1. Open script.js
2. Change the values inside eventDetails:
   - parentsName
   - eventDate
   - eventTime
   - eventVenue
3. Save and redeploy.

How to test:
Open index.html in your browser, fill RSVP, click Submit, then check your Google Sheet.

How to host on Vercel:
1. Go to vercel.com
2. Create a free account
3. Create New Project
4. Upload this folder or connect GitHub
5. Deploy
6. Share the final Vercel link on WhatsApp.
